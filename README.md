# Farhanulla Premium Portfolio

Run locally:

```bash
pip install -r requirements.txt
python3 app.py
```

Open http://127.0.0.1:5000

This project includes:
- smooth loader
- custom cursor
- 3D animated background
- parallax hero
- flip hover project cards
- project detail pages
- smart gateways
- NLP-inspired search
- data science insight cards
- custom monogram logo
